# What is it? #

**ProSE (Problem Search Environment)** is a open source standard environment developed for teaching Artificial Intelligence search algorithms in C++. It is a search engine capable to work with all AI related problems in a single environment. You can use a modeled problem or can easily model new ones. ProSE current version is developed in DevC++ for windows. Just download, compile and run.

## What problems are ready for use? ##

Currently you can find a computational environment for the following problems:

  * Water-jugs problem
  * Tower of Hanoi
  * Rubik's cube
  * Oil and tubes problem

## What are the available search algorithms? ##

ProSE also implements the following search algorithms:

  * Depth First Search
  * Breadth First Search
  * A`*` Search
