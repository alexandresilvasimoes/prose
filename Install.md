# Instalation Guide #

In order to install ProSE you will need:

  1. Download latest version of prose [here](http://code.google.com/p/prose/downloads/list)
  1. Download DevC++ [here](http://www.bloodshed.net/devcpp.html)
  1. Select your problem: in folder "/problems/my\_problem" you will find four files:
    1. ProblemData.h
    1. ProblemRules.h
    1. ProblemData.cpp
    1. ProblemRules.cpp
> > Copy these files to the main folder, overwriting previous ones.
  1. Compile (rebuild all) and run.


> Since ProSE is written in C++, it can be compiled in any Operational Systems. However, current Graphical User Interface (GUI) -- available in main.cpp -- requires Microsoft Windows. Next versions will allow cross-platform compilation.
